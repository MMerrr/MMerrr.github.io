Important Twitch Mod Commands

/monitor <username> | Monitors the specified user for chat activity.
/unmonitor <username> | Stops monitoring the specified user.
/restrict <username> | Restricts the specified user from certain actions.
/unrestrict <username> | Removes restrictions from the specified user.
/timeout <username> [seconds] | Temporarily blocks a user from chatting for the specified duration (to lift timeout, set it to 1 second or use the /unban command).
/ban <username> | Permanently bans the specified user from chatting.
/unban <username> | Unbans the specified user, allowing them to chat again.
/poll | Initiates a poll for viewers to vote on.
/endpoll | Ends the current poll.
/deletepoll | Deletes the current poll.
/requests | Displays channel point requests.

Situational Twitch Mod Commands

/clear | Clears the entire chat, useful for dealing with questionable spam (individual message deletion is usually preferred).
/slow <seconds> | Sets a time limit for users to send consecutive messages.
/slowoff | Disables slow mode in the chat.
/followers <time> | Restricts chat participation based on how long users have been following (e.g., 30m = 30 minutes, 2h = 2 hours, 2d = 2 days, 2w = 2 weeks, 2mo = 2 months).
/followersoff | Disables follower-only mode.
/subscribers | Restricts chat to subscribers only.
/subscribersoff | Disables subscribers-only mode.
/uniquechat | Requires messages to be 9 characters and unique in chat.
/uniquechatoff | Disables unique chat mode.
/emoteonly | Allows only emotes in chat (100% emotes).
/emoteonlyoff | Disables emote-only mode.
